# Web Recorder

Web Recorder 是一款功能强大的 Chrome 浏览器插件，专门用于录制当前标签页（Tab）的内容并生成视频文件。

## 核心功能

*   **当前 Tab 录制**：仅录制选中的标签页可视区域，不干扰其他 Tab 或系统桌面。
*   **多模式录制**：
    *   **一直录制模式**：手动开始与结束，支持自定义最大录制时长（默认 60 分钟）。
    *   **限时录制模式**：设定时长后自动停止。
*   **灵活控制**：支持录制过程中的暂停、继续与手动结束。
*   **后台运行**：即使切换 Tab 或浏览器最小化，录制也会持续进行。
*   **本地保存**：录制结束后自动生成含时间戳的 WebM 视频文件并保存到本地。

## 安装说明

1.  下载本项目代码。
2.  打开 Chrome 浏览器，进入 `chrome://extensions/`。
3.  开启右上角的“开发者模式”。
4.  点击“加载已解压的扩展程序”，选择本项目所在的文件夹。

## 使用指南

1.  在需要录制的页面点击项目图标。
2.  选择录制模式（一直录制或设定时长）。
3.  点击“开始录制”。
4.  在弹出窗口或插件面板中控制录制状态（暂停/结束）。
